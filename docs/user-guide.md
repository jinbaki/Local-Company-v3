# 사용자 가이드

V3에서 사용자는 Local Company 웹앱이 아니라 Codex와 대화한다.

## 기본 사용

1. Local Company V3를 실행한다.
2. Codex에서 작업을 말한다.
3. Codex가 MCP로 PM에게 작업을 위임한다.
4. PM이 계획을 만들고 worker가 산출물을 만든다.
5. Codex가 완료 또는 대표 확인 필요 알림을 알려준다.
6. 사용자는 Codex와 함께 산출물을 검토한다.

## 예시

```text
Codex, 이 문서를 Local Company V3 PM에게 맡겨서
실행 가능한 산출물 3개로 만들어줘.
완료되면 알려줘.
```

## 상황판

웹 상황판에서는 다음을 확인한다.

- active run
- PM 계획 요약
- 작업 큐
- worker 상태
- 산출물 목록
- 대표 확인 항목
- 최근 알림

PM 채팅창은 없다. PM에게 지시하려면 Codex에게 말한다.

## 산출물 확인

Codex에게 이렇게 요청한다.

```text
Local Company V3에서 방금 완료된 산출물 목록 보여줘.
첫 번째 산출물 내용을 요약해줘.
```

## 대표 확인 항목

비용, 외부 연락, 법무, 공개 약속, 의료/투자 판단처럼 사람이 정해야 하는 항목은 run을 멈추고 대표 확인 항목으로 올라온다.

답변은 Codex에게 말하면 된다.

```text
첫 번째 대표 확인 항목은 PM 추천안대로 진행하라고 답변해줘.
```
