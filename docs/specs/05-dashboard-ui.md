# V3 Dashboard UI 명세

V3 웹앱은 상황판이다. PM 채팅 앱이 아니다.

## 첫 화면

첫 화면은 active run 상황판이다.

표시 항목:

- Local Company 실행 상태
- Codex CLI 연결 상태
- active run 목록
- 최근 알림
- 대표 확인 필요 항목

## Run 상세

Run 상세는 다음 섹션으로 구성한다.

1. Run 요약
   - 제목
   - 상태
   - 진행률
   - 시작/종료 시각
   - 현재 병렬 worker 수

2. PM 계획
   - PM 요약
   - 작업 수
   - 산출물 수
   - worker 역할
   - 대표 결정 필요 여부

3. 작업 큐
   - task 제목
   - 담당 worker
   - 상태
   - 재시도 횟수
   - 연결 산출물

4. Worker 실행
   - worker run 상태
   - 모델
   - 시작/종료 시각
   - 실패 이유

5. 산출물
   - 산출물명
   - 상태
   - 최신 버전
   - PM 리뷰 요약

6. 대표 확인 항목
   - 결정 제목
   - 이유
   - PM 추천
   - 막고 있는 task/artifact

7. 이벤트 타임라인
   - run 시작
   - worker 시작/완료
   - 산출물 생성
   - PM 리뷰
   - 결정 요청
   - 완료

## 금지되는 UI

- PM 채팅 입력창
- "PM에게 메시지 보내기" 버튼
- "작업 하나 실행" 중심 버튼
- 인계 보고서 버튼
- 사람이 직접 task를 만드는 복잡한 폼

## 허용되는 안전 UI

다음은 상황판에서도 허용된다.

- 실행 중지
- Codex 연결 확인
- 산출물 열기
- 로그 보기
- 대표 확인 항목 복사

대표 결정 답변은 기본적으로 Codex를 통해 입력한다. Dashboard에 입력 UI를 둘지는 후순위로 둔다.
